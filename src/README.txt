
ISTRUZIONI PER AVVIARE NINETEEN10 MULTIPLAYER (locale)

1. Assicurati di avere Node.js installato.
2. Vai nella cartella del progetto con il terminale.
3. Installa le dipendenze:
   npm install
4. Avvia il server:
   npm start
5. Apri due browser su:
   http://localhost:3000
   Inserisci due nickname diversi per giocare.
